##  🔐 Trabalho Prático – Sistema de Login em Java (Swing)

## 🎯 **Objetivo**

Desenvolver uma aplicação em **Java utilizando Swing** que implemente uma **tela de login funcional**, aplicando conceitos de:

- Interface gráfica (GUI)
- Programação Orientada a Objetos (POO)
- Validação de dados
- Organização de código

---

## 📚 **Base do Trabalho**

O sistema deve ser desenvolvido com base nos materiais:

- 📄 Login Funcional
- 📄 Tela de Login

(Disponibilizados no Notion pelo professor)

---

## 🧩 **Descrição do Projeto**

Você deverá criar um sistema com uma **tela de login**, onde o usuário informa:

- 👤 Usuário
- 🔒 Senha

O sistema deve validar essas informações e apresentar o comportamento adequado.

---

## ⚙️ **Requisitos Funcionais**

### 🔐 Tela de Login

A interface deve conter:

- Campo de texto para **usuário**
- Campo de senha (**JPasswordField**)
- Botão **Entrar**

---

### ✅ Funcionalidade do Botão "Entrar"

Ao clicar no botão:

- O sistema deve:
    - Verificar se os campos estão preenchidos
    - Validar usuário e senha

### 🔎 Regras de validação:

- Usuário padrão: `admin`
- Senha padrão: `123`

### 📌 Comportamento esperado:

| Situação | Resultado |
| --- | --- |
| Campos vazios | Exibir mensagem de erro |
| Dados incorretos | Exibir mensagem "Usuário ou senha inválidos" |
| Dados corretos | Exibir mensagem "Login realizado com sucesso" |

---

## 🎨 **Requisitos de Interface**

- Utilizar **JFrame**
- Centralizar a tela
- Utilizar **JLabel, JTextField, JPasswordField e JButton**
- Interface organizada e alinhada
- Título da janela: `Sistema de Login`

---

## 🧠 **Requisitos Técnicos**

- Utilizar **classes separadas (boa prática de organização)**
- Utilizar **métodos para validação**
- Aplicar conceitos de:
    - Encapsulamento
    - Organização de código
- Código deve estar **comentado**

---

## 🚀 **Requisitos Extras (Diferencial)**

*(Opcional, mas aumenta nota)*

- Mostrar quantidade de tentativas de login
- Bloquear acesso após 3 tentativas
- Criar uma nova tela (ex: "Bem-vindo")
- Estilizar melhor a interface (cores, fontes, etc.)

---
